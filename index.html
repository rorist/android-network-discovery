Network Discovery for Android


Android network tool: Discover hosts and scan their ports in your Wifi network. A great tool for testing your network security.

  • Project on github (source code)
  • Download APK
  • Found for free on the Android Market, SlideMe.org, AppsLib and Insyde Market
If you like this app, you can buy me a beer:

Features

  • Discover Machines on a Network
  • TCP Port Scanner
  • NIC vendor database
  • Service banner retrieval
  • Service detection according to the service banner
  • Connect to known services (http, ssh, telnet, ftp, ...) directly from the application results
  • Export results to your sdcard in XML
  • Fast access to Wifi Settings
  • Adaptive scanning rate (slow start, then adaptive to network latency)
  • Mobile network (3g) scans, even if it's not relevant
  • Open Source

Future releases

  • Open port OS Fingerprint
  • Root features: SYN port scan, and other improvements or additions. But first I really want the non-root version to be complete.
  • Nmap wrapper
  • Metasploit wrapper
